Magellan

Magellan is based on the similar projects magbath and MODMAG. The two aforementioned projects are not free software so magellan is their free software counterpart.

Additionally, magellan is meant to enable users, both researchers and programmers to more easily modify source code and improve the software. Python was chosen as the programming language to make the code cleaner and easier to read.

The user interface is designed to enable better control and input of parameters and data, without any boundaries, through the use of commandline and input plain text files.

Magellan developer site

Development of Magellan is organized through the Savannah repository.
